# Todo App

Una aplicación simple de tareas construida con Spring Boot.

## Características
- Crear tareas
- Marcar tareas como completadas
- Eliminar tareas

## Requisitos
- Java 11+
- Maven

## Instalación
```bash
mvn clean install
```

## Ejecución
```bash
mvn spring-boot:run
```

## Uso
La aplicación estará disponible en `http://localhost:8080`
o el puerto seleccionado en `src/main/resources/application.properties`

## Tecnologías
- Spring Boot
- Spring Data JPA
